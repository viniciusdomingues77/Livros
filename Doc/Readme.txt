0) Criar um banco vazio no SQLServer 2017 developer ou superior chamado Livros 
1)Executar o arquivo livros.sql contra o bd criado 
2) O usuário [user_livro] com password [123456] deverá ser criado junto com os scripts
conceda grant com ouwner da base de dados Livros
3) o método ConectaBancoLivros deverá ter a string de conexão ajustada para 
o BD host correto
4) O projeto acompanha a pasta com o visual basic6 e o service pack 6 caso não tenha ainda instalado
5) O conteúdo da pasta Arquivos deve ser copiado e colado na win32,em seguida o componente
VBPrnDlg.dll deve ser registrado
6) Dentro da pasta Prints estão as referencias e componentes a serem setados dentro do ambiente 
do Visual Basic 6
7) Dentro da pasta apresentação está um video com o funcionamento básico do projeto.
